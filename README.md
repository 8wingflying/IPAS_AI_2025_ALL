# 初級 能力鑑定
## 科目1: L11 人工智慧基礎概論
  - 能掌握人工智慧的基本概念與應用領域，瞭解人工智慧的定義、基本理論和核心目標，及 AI技術在隱私、安全性與倫理層面可能面臨的挑戰。
  - 具備基本的資料處理知識，包括資料收集、清洗、分析和視覺化等過程，並能夠理解資料在 AI模型中扮演的角色與作用 。
  - 理解機器學習的基本原理及其在 AI中的作用，並熟悉常見的機器學習方法及其應用。
  - 能夠區分鑑別式 AI和生成式 AI的特性，並理解生成式 AI的概念 、 基本原理及其相關應用。
